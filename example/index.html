Getting Started

An AngularJS state-based router designed for flexibility and ease of use.

StateRouter is designed to be used in a modular integration with:

  • StateRouter
  • StateView
  • StateLoadable

Installation

To install in your project, simply install from npm in your project folder

$ cd my_project
$ npm install angular-state-router --save

Then include the script in your <head> tag

<script src="/node_modules/angular-state-router/dist/state-router.min.js"></script>

Definition

Set up application, typically during the configuration phase, through $stateProvider provider

angular.module('myApp', ['angular-state-router'])
  .config(function($stateProvider) {

    $stateProvider

      // Define states
      .state('landing', {
        url: '/'
      })

      // Set initialization location; optionally
      .init('landing');

  });

Use

Access application state through $state service

angular.module('myApp')
  .controller('MyController', function($scope, $state) {
  
    // Listen to events
    $state.on('change:complete', function() {

      // Get the current state
      var current = $state.current();

      console.log(current.name);
    });

    $scope.onClickMarketingCampaign = function() {

      // Request changes
      $state.change('catalogs.products.items', { id: 'e239' });

    };

  });
